Command: clear

Description: 'clear' will refresh the terminal screen.

Enter the command [ clear ]now to clear the screen.

Command: ls

'Description: ls' will diplay the contents of a directory that are not hidden.

Enter the command [ ls ] now to list the contents in the directory.

Command: mkdir {directory name}

Description: 'mkdir' makes a directory/folder.

For this tutorial we will be making a directory called ltDir.

Enter the command [mkdir ltDir] now to create a new directory.

Command: cd {directory name}

Description: 'cd' will change directories. This command is how you will navigate around the terminal.

Enter the command [cd ltDir] now to to move into the new directory you just created.
